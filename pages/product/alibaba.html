云服务器 ECS

云服务器 ECS 概述

云服务器（Elastic Compute Service，简称 ECS）是一种简单高效、处理能力可弹性伸缩的计算服务，帮助您快速构建更稳定、安全的应用，提升运维效率，降低 IT 成本，使您更专注于核心业务创新。

云服务器 ECS 功能

丰富的实例规格，支持随时升级，满足各种业务需求

  • 云服务器 ECS 提供丰富的实例规格（CPU、内存）和带宽、存储盘选择，还可随时不停机升级带宽，5 分钟内停机升级 CPU 和内存，支撑业务的持续发展。99.95% 的高可用性，为业务的稳定运行提供保障。 对于高并发读写和高性能需求，还可选择 I/O 优化实例 。其对底层软硬件都进行了升级和调整，挂载 SSD 云盘、高效云盘时能够获得全部性能。对于不支持 I/O 优化的实例，挂载 SSD 云盘、高效云盘时，通常最高仅可获得 1000 左右的 IOPS 性能。

实例系列 II，硬件升级，提供更强大的计算能力和更大的内存可选范围

  • ECS 推出新一代的实例系列，原来线上售卖的实例称为实例系列 I（以下称系列 I），而新实例系列为实例系列 II（以下称系列 II）。 相比较系列 I 采用的 Intel Xeon CPU、DDR3 内存，全新硬件升级的实例系列 II， 采用 Intel Xeon E5-2680 v3（Haswell）CPU、DDR4 内存，并默认为 I/O 优化实例，同时增加了一些新的指令集，使整数和浮点运算的性能翻倍，整体计算能力更强。

支持 4 种数据存储盘，满足不同的 I/O 性能要求

  • 用户可为 ECS 实例搭配数据块存储，以获得持久性、高可靠的数据块级的随机存储。像对待硬盘一样，用户可以对挂载到 ECS 实例上的数据块存储做格式化、创建文件系统等操作，并对数据做持久化存储。 根据不同应用场景对 I/O 性能的需求，我们提供 4 种不同性能的数据块存储，既可以单独使用、也可以组合使用。

支持自动 / 自定义备份，快速恢复数据

  • 用户可以在控制台设置自动快照策略，对云服务器的系统盘、磁盘数据生成快照，也可以通过控制台或 OpenAPI 手动创建快照。通过快照回滚，可以快速恢复快照生成时间点的数据状态，加强数据安全。 目前每个磁盘最多可创建 64 个快照。

提供控制台、远程终端和 API 等多种管理方式，给您完全管理权限

  • 用户对云服务器拥有完全控制权，您可以像管理传统服务器一样，通过远程连接终端，解决系统问题，进行各项操作；也可以通过管理控制台、API、命令行等类似工具对服务器进行重启、重置登陆密码、网络隔离等操作。

提供公网 IP 及私网 IP，用于不同的连接场景

  • 公网 IP：如果您在购买实例时选择了大于 0M 的公网带宽，实例创建时会为您分配一个公网 IP。公网 IP 可以用于实例与 Internet 之间互访，也可以用于实例与云服务之间互访。
  • 私网 IP :可以用于 ECS 实例之间内网互访，可以用于 ECS 实例与其他云服务之间内网互访，如 负载均衡 SLB、开放存储服务 OSS、数据库 RDS。同一地域内实例之间通过私网 IP 进行的通讯流量是免费的。（注：不同地域之间的产品内网 IP 不互通）

专有网络和经典网络，提供灵活的网络规划选择

  • 经典网络：IP 地址由阿里云统一分配，配置简便。在经典网络下，用户可以通过安全组设置单台或多台云服务器的网络访问控制，进行安全隔离。了解安全组的详细信息 >>
  • 专有网络：可以让用户构建出一个隔离的网络环境， 并可以完全掌控自己的网络，不仅能使用安全组进行访问控制，还能选择自有 IP 地址范围、划分网段、配置路由表和网关等。此外，也可以通过专线 / VPN 等连接方式将 VPC 与传统数据中心组成一个按需定制的网络环境，实现应用的平滑迁移上云。

免费开通云盾，提供网络安全、服务器安全等基础防护

  • DDoS 基础防护：提供最高 5G 的 DDoS 防护能力，可防御 SYN flood、UDP flood、ICMP flood、ACK flood 常规 DDoS 攻击。了解详情 >>
  • 服务器安全功能（安骑士）：包含暴力破解密码拦截、木马查杀、异地登录提醒、高危漏洞修复的防入侵功能。

免费提供云监控，并支持多种实时预警

  • 站点监控：提供对 http、ping、dns、tcp、udp、smtp、pop、ftp 等服务的可用性和响应时间的统计、监控、报警服务。
  • 云服务监控：提供对云服务的监控报警服务，对用户开放自定义监控的服务，允许用户自定义个性化监控需求。
  • 报警及联系人管理：提供对报警规则，报警联系人的统一、批量管理服务。支持多报警方式：短信、邮件、旺旺、接口回调。

丰富的镜像，免安装快速部署操作系统与软件

  • 镜像（Image）是 ECS 实例运行环境的模板，一般包括操作系统和预装的软件。用户使用镜像创建 ECS 实例和更换 ECS 实例的系统盘，进行免安装快速部署。我们提供 3 种镜像的途径：
    • 阿里云官方提供的公共镜像：提供 Windows、Linux 操作系统的多个发行版本。
    • 去云市场选择第三方服务商（ISV）提供的镜像：丰富的服务器运行环境、工具和软件镜像。 去云市场逛逛>>
    • 共享镜像：自定义镜像的拥有者可以通过 ECS 控制台或者 ECS API 主动把镜像共享给其他用户，被共享用户可以使用其他账号共享的镜像创建 ECS 实例和更换系统盘。
